What Would a Top-Tier Consultant Do?

McKinsey/Bain/BCG Approach:

  1. Baseline Measurement (Week 1):
    • Shadow 10 sales reps for 1 week
    • Time every SF interaction (login, query, export, navigation)
    • Document frequency and duration
    • Establish true baseline
  2. Implement Analytics (Week 2):
    • Add usage logging to GTM-Brain
    • Track: Who queries what when, success/failure, query type
    • Build dashboard showing actual adoption
  3. A/B Testing (Month 1-2):
    • Control group: 20 users without GTM-Brain access
    • Test group: 20 users with access
    • Measure: Time spent in SF, deal velocity, forecast accuracy
    • Compare groups
  4. Triangulate Value (Month 3):
    • Usage data (actual queries × measured time savings)
    • A/B test results (control vs test group difference)
    • User surveys (self-reported time savings)
    • Converge on defensible number

Consultant's Defensible Range: $150K - $250K/year with 6 months of data
